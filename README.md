A sleek and efficient currency converter built with React and TypeScript. This application allows users to seamlessly convert amounts between multiple currencies using real-time exchange rates fetched from a public API. Perfect for travelers, business professionals, and developers looking to understand currency exchange mechanisms.

## Features:
- Real-Time Conversion: Fetches up-to-date exchange rates for accurate conversions.
- Multi-Currency Support: Convert between popular currencies like USD, EUR, GBP, JPY, AUD, CAD, and PKR.
- Dynamic Calculations: Instantly calculates converted amounts as inputs are updated.
- User-Friendly Interface: Intuitive design ensures ease of use.
- Built with TypeScript: Ensures type safety and robust development.

## Tech Stack:
- **React**: Frontend framework for dynamic UIs.
- **TypeScript**: Enhances code reliability and maintainability.
- **Fetch API**: Handles real-time API requests.
